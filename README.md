This is the github repo for "Superoxide dismutase shapes manganese stoichiometry in Southern Ocean diatoms".

Data for the scripts is all described in the manuscript and submitted in Zenodo.

Top re-run the analyses, you'll need to get the correct file structure (all raw files are uploaded in the Zenodo submission https://doi.org/10.5281/zenodo.19614333). Specifically, you'll need:

amundsen-prot/
├── data/
│   ├── intermediate_data/
│   └── Amudsen_SRM_data_2021/
└── scripts/

All files from Zenodo should be in data, except total_taxonomix_biomass_per_filter.csv, total_taxonomic_biomass_no_unambiguous_renorm.csv, and pride_submission_metadata.csv should be in 'intermediate_data'. Sample_SRM_export_01.xlsx should be in 'Amudsen_SRM_data_2021' subfolder. All scripts from this repo should be in scripts/.
